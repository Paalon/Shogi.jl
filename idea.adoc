== DataFrames

特定の列を削除する。

[source, julia]
----
select(df, Not(:column_name))
----

列を追加する。

[source, julia]
----
insertcols!(df, :column_name => column)
----

表を結合する。

[source, julia]
----
hcat(df1, df2)
----
